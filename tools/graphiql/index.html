О проекте

graphiql — клиенты API, шлюзы, webhooks и обмен данными между системами. Связывается с другими сервисами. Открытый исходный код.

graphiql: graphiQL & the GraphQL LSP Reference Ecosystem for building browser & IDE tools. In this catalog (Infrastructure), listed as “API & интеграции”: API clients, gateways, webhooks, ETL, data pipelines.
