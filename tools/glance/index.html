О проекте

glance — админки, панели и внутренние приложения для команды без долгой разработки. Можно развернуть на своём сервере. Ключевые темы: Aggregator, Дашборд, Feed Reader, Homelab, Homepage и Мониторинг. Проект: бесплатный open-source проект, на GitHub 36k+ звёзд и основной стек — Go. Открытый исходный код.

glance is a на своём сервере dashboard that puts all your feeds in one place. In this catalog (Business), listed as “Business Low-Code”: Internal tools, admin panels, dashboard builders for teams. Key topics: Aggregator, Dashboard, Feed Reader, Homelab, Homepage, and Monitoring. Project details: free с открытым кодом, 36k+ GitHub stars, and built mainly in Go.
