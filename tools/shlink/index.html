О проекте

shlink — email-рассылки, формы, опросы и продвижение. Можно развернуть на своём сервере. Ключевые темы: Laminas, Mezzio, Php, REST API и Url Shortener. Проект: бесплатный с открытым кодом проект, на GitHub 5k+ звёзд и основной стек — PHP. Связывается с другими сервисами.

shlink is the definitive на своём сервере URL shortener. In this catalog (Business), listed as “Marketing & Email”: Email campaigns, marketing автоматизация, SEO, social scheduling. Key topics: Laminas, Mezzio, Php, REST API, and Url Shortener. Project details: free с открытым кодом, 5k+ GitHub stars, and built mainly in PHP.

В каталоге отнесён к разделам: «Маркетинг и рассылки» — email-рассылки, формы, опросы и продвижение; «API и интеграции» — клиенты API, шлюзы, webhooks и обмен данными между системами.
