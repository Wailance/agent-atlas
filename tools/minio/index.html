О проекте

minio — хранение данных, SQL-клиенты и админки баз. Ключевые темы: Amazon S3, Cloud, Cloudnative, Cloudstorage, K8s и Multi Cloud. Проект: бесплатный с открытым кодом проект, на GitHub 61k+ звёзд и основной стек — Go.

minio: minIO is a high-performance, S3 compatible object store, open sourced under GNU AGPLv3 license. In this catalog (Infrastructure), listed as “Databases & Admin”: SQL clients, database admin, vector DB, time-series. Key topics: Amazon S3, Cloud, Cloudnative, Cloudstorage, K8s, and Multi Cloud. Project details: free с открытым кодом, 61k+ GitHub stars, and built mainly in Go.

В каталоге отнесён к разделам: «Базы данных» — хранение данных, SQL-клиенты и админки баз; «API и интеграции» — клиенты API, шлюзы, webhooks и обмен данными между системами.
