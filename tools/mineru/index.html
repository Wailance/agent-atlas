О проекте

MinerU — собирает данные с сайтов, PDF и документов — вручную или с помощью ИИ. Автоматизирует процессы и связывает сервисы, работает с ИИ-агентами. Ключевые темы: Ai4science, Document Analysis, Docx, Extract Data, Layout Analysis и Распознавание текста. Подходит, если нужно: автоматизирует процессы и связывает сервисы, работает с ИИ-агентами и обрабатывает документы и контент для ИИ. Проект: бесплатный open-source проект, на GitHub 75k+ звёзд и основной стек — Python. Открытый исходный код.

MinerU: transforms complex documents like PDFs and Office docs into языковые модели-ready markdown/JSON for your Agentic рабочие процессыs. In this catalog (AI & Agents), listed as “Web Scraping”: Web data collection, crawlers, AI scraping, document parsing. Key topics: Ai4science, Document Analysis, Docx, Extract Data, Layout Analysis, and OCR. Useful when you need to automate рабочие процессыs, build AI agents, and work with documents and поиск по документам. Project details: free с открытым кодом, 75k+ GitHub stars, and built mainly in Python.

В каталоге отнесён к разделам: «Парсинг и сбор данных» — собирает данные с сайтов, PDF и документов — вручную или с помощью ИИ; «Поиск по документам» — подключает ваши файлы и базы к чату с ИИ, чтобы ответы были точнее.

На GitHub у проекта более 75k+ звёзд — это один из заметных репозиториев в своей нише.
