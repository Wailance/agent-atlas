О проекте

qdrant — подключает ваши файлы и базы к чату с ИИ, чтобы ответы были точнее. Поддерживает ИИ-агентов. Открытый исходный код.

qdrant: high-performance, massive-scale Vector Database and Vector Search Engine for the next generation of AI. Also available in the cloud https://cloud.qdrant.io/. In this catalog (AI & Agents), listed as “поиск по документам / Context”: поиск по документам, memory and context for языковые модели. Key topics: Ai Search, Ai Search Engine, Embeddings Similarity, Hnsw, Hybrid Search, and Image Search. Project details: free с открытым кодом, 33k+ GitHub stars, and built mainly in Rust.
