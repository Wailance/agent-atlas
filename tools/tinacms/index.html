О проекте

tinacms — aPI-first CMS, контентные модели и headless-управление сайтом. Обрабатывает документы и контент для ИИ, есть визуальный или low-code интерфейс. Ключевые темы: CMS, Content Management System, Forestry, Markdown, Nextjs и React. Подходит, если нужно: обрабатывает документы и контент для ИИ и есть визуальный или low-code интерфейс. Проект: есть бесплатный tier и платные возможности, на GitHub 14k+ звёзд и основной стек — TypeScript. Открытый исходный код.

В каталоге отнесён к разделу: «Headless CMS» — aPI-first CMS, контентные модели и headless-управление сайтом.

На GitHub у проекта более 14k+ звёзд — это один из заметных репозиториев в своей нише.
