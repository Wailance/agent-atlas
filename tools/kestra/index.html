О проекте

kestra — dAG, ETL, расписания и пайплайны данных для надёжного запуска задач. Автоматизирует процессы и связывает сервисы, работает с ИИ-агентами. Ключевые темы: ИИ-агенты, Автоматизация, Control Plane, Data Engineering, Data Orchestration и Data Orchestrator. Подходит, если нужно: автоматизирует процессы и связывает сервисы и работает с ИИ-агентами. Проект: есть бесплатный tier и платные возможности, на GitHub 27k+ звёзд и основной стек — Java.

В каталоге отнесён к разделам: «Оркестрация данных» — dAG, ETL, расписания и пайплайны данных для надёжного запуска задач; «API и интеграции» — клиенты API, шлюзы, webhooks и обмен данными между системами.

На GitHub у проекта более 27k+ звёзд — это один из заметных репозиториев в своей нише.
