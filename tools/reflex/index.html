О проекте

reflex — собирает ИИ-приложения и сервисы без глубокого программирования. Ключевые темы: Дашборд, Data Analysis, Data Science, Data Visualization, Для разработчиков и Framework. Проект: есть бесплатный tier и платные возможности, на GitHub 29k+ звёзд и основной стек — Python. Поддерживает ИИ-агентов.

reflex: 🕸️ Web apps in pure Python 🐍. In this catalog (AI & Agents), listed as “App Builders”: AI app builders and no-code/low-code platforms. Key topics: Dashboard, Data Analysis, Data Science, Data Visualization, Developer tools, and Framework. Project details: freemium model, 29k+ GitHub stars, and built mainly in Python.
