О проекте

DeerFlow — запускает несколько ИИ-агентов и связывает их в цепочки задач. Работает с ИИ-агентами, обрабатывает документы и контент для ИИ. Ключевые темы: Multi Agent, Langgraph, Superagent, Агент, Agentic и Agentic Framework. Подходит, если нужно: работает с ИИ-агентами и обрабатывает документы и контент для ИИ. Проект: бесплатный с открытым кодом проект, на GitHub 77k+ звёзд и основной стек — Python. Связывается с другими сервисами.

DeerFlow is an с открытым кодом long-horizon SuperAgent harness that researches, codes, and creates. With the help of sandboxes, memories, tools, skill, subagents and message gateway, it handles different levels of tasks that could take minutes to hours. In this catalog (AI & Agents), listed as “Agent Harness”: Orchestration, несколько ИИ-агентов рабочие процессыs, long-horizon tasks. Key topics: Multi Agent, Langgraph, Superagent, Agent, Agentic, and Agentic Framework. Project details: free с открытым кодом, 77k+ GitHub stars, and built mainly in Python.

В каталоге отнесён к разделу: «Оркестрация агентов» — запускает несколько ИИ-агентов и связывает их в цепочки задач.
