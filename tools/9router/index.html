О проекте

9router — один вход для разных ИИ-моделей: маршрутизация, лимиты, ключи. Работает с ИИ-агентами. Ключевые темы: Proxy, Claude, Cursor, ИИ-агенты, Ai Gateway и Anthropic. Проект: бесплатный с открытым кодом проект, на GitHub 22k+ звёзд и основной стек — JavaScript.

9router: unlimited FREE AI coding. Connect Claude Code, Codex, Cursor, Cline, Copilot, Antigravity to FREE Claude/GPT/Gemini via 40+ providers. Auto-fallback, RTK -40% tokens, never hit limits. In this catalog (AI & Agents), listed as “языковые модели Gateways”: Routing, proxies and unified API for языковые модели providers. Key topics: Proxy, Claude, Cursor, AI agents, Ai Gateway, and Anthropic. Project details: free с открытым кодом, 22k+ GitHub stars, and built mainly in JavaScript.
