О проекте

generator-jhipster — админки, панели и внутренние приложения для команды без долгой разработки. Ключевые темы: Angular, Cloud, Generator и Jhipster. Проект: бесплатный с открытым кодом проект, на GitHub 22k+ звёзд и основной стек — TypeScript.

generator-jhipster: jHipster is a development platform to quickly generate, develop, & deploy modern веб-приложения & microservice architectures. In this catalog (Business), listed as “Business Low-Code”: Internal tools, admin panels, dashboard builders for teams. Key topics: Angular, Cloud, Generator, and Jhipster. Project details: free с открытым кодом, 22k+ GitHub stars, and built mainly in TypeScript.
