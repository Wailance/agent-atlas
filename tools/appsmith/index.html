О проекте

appsmith — админки, панели и внутренние приложения для команды без долгой разработки. Автоматизирует процессы и связывает сервисы. Ключевые темы: Admin Dashboard, Admin Panels, App Builder, Автоматизация, Crud и Custom Internal. Проект: есть бесплатный tier и платные возможности, на GitHub 40k+ звёзд и основной стек — TypeScript.

appsmith: platform to build admin panels, internal tools, and dashboards. Integrates with 25+ databases and any API. In this catalog (Business), listed as “Business Low-Code”: Internal tools, admin panels, dashboard builders for teams. Key topics: Admin Dashboard, Admin Panels, App Builder, автоматизация, Crud, and Custom Internal. Project details: freemium model, 40k+ GitHub stars, and built mainly in TypeScript.
