О проекте

v2rayNG — другие полезные open-source проекты. Открытый исходный код.

v2rayNG is a V2Ray client for Android, support Xray core and v2fly core. In this catalog (Infrastructure), listed as “General OSS”: General с открытым кодом software. Key topics: Android, Proxy, Shadowsocks, Socks5, Trojan, and V2fly. Project details: free с открытым кодом, 60k+ GitHub stars, and built mainly in Kotlin.
