О проекте

agentops — следит за качеством ответов, ошибками и расходами ИИ-приложений. Автоматизирует процессы и связывает сервисы, работает с ИИ-агентами. Ключевые темы: Агент, Agentops, Agents Sdk, ИИ, Anthropic и Autogen. Подходит, если нужно: автоматизирует процессы и связывает сервисы и работает с ИИ-агентами. Проект: бесплатный open-source проект, на GitHub 6k+ звёзд и основной стек — Python. Открытый исходный код.

agentops: python SDK for AI agent monitoring, языковые модели cost tracking, benchmarking, and more. Integrates with most языковые модели and agent frameworks including CrewAI, Agno, OpenAI Agents SDK, Langchain, Autogen, AG2, and CamelAI. In this catalog (AI & Agents), listed as “Observability / Eval”: Tracing, evaluation, monitoring of AI applications. Key topics: Agent, Agentops, Agents Sdk, AI, Anthropic, and Autogen. Project details: free с открытым кодом, 6k+ GitHub stars, and built mainly in Python.
