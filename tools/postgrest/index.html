О проекте

postgrest — клиенты API, шлюзы, webhooks и обмен данными между системами. Ключевые темы: API, Automatic Api, База данных, Haskell, Http и Pg. Проект: бесплатный с открытым кодом проект, на GitHub 27k+ звёзд и основной стек — Haskell. Связывается с другими сервисами.

postgrest: rEST API for any Postgres database. In this catalog (Infrastructure), listed as “API & интеграции”: API clients, gateways, webhooks, ETL, data pipelines. Key topics: API, Automatic Api, Database, Haskell, Http, and Pg. Project details: free с открытым кодом, 27k+ GitHub stars, and built mainly in Haskell.
