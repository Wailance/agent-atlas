О проекте

Mastra — запускает несколько ИИ-агентов и связывает их в цепочки задач. Работает с ИИ-агентами. Ключевые темы: Процессы, Наблюдаемость, Production, Агенты, ИИ и Чат-боты. Проект: есть бесплатный tier и платные возможности, на GitHub 26k+ звёзд и основной стек — TypeScript.

Mastra: mastra is the modern TypeScript framework for AI-powered applications and agents. In this catalog (AI & Agents), listed as “Agent Harness”: Orchestration, несколько ИИ-агентов рабочие процессыs, long-horizon tasks. Key topics: рабочие процессыs, Observability, Production, Agents, AI, and Chatbots. Project details: freemium model, 26k+ GitHub stars, and built mainly in TypeScript.

В каталоге отнесён к разделу: «Оркестрация агентов» — запускает несколько ИИ-агентов и связывает их в цепочки задач.
