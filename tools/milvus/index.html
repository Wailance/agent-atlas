О проекте

milvus — подключает ваши файлы и базы к чату с ИИ, чтобы ответы были точнее. Поддерживает ИИ-агентов. Открытый исходный код.

milvus: milvus is a high-performance, cloud-native vector database built for scalable vector ANN search. In this catalog (AI & Agents), listed as “поиск по документам / Context”: поиск по документам, memory and context for языковые модели. Key topics: Anns, Cloud Native, Diskann, Distributed, Embedding Database, and Embedding Similarity. Project details: free с открытым кодом, 45k+ GitHub stars, and built mainly in Go.
