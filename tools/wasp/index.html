О проекте

wasp — собирает ИИ-приложения и сервисы без глубокого программирования. Ключевые темы: Node, Prisma, React, Wasp и Web App. Проект: есть бесплатный tier и платные возможности, на GitHub 19k+ звёзд и основной стек — TypeScript. Поддерживает ИИ-агентов.

wasp is the batteries-included full-stack framework for the AI era. Develop JS/TS web apps (React, Node.js, and Prisma) using declarative code that abstracts away complex full-stack features like auth, background jobs, RPC, email sending, end-to-end type safety, single-command deployment, and more. In this catalog (AI & Agents), listed as “App Builders”: AI app builders and no-code/low-code platforms. Key topics: Node, Prisma, React, Wasp, and Web App. Project details: freemium model, 19k+ GitHub stars, and built mainly in TypeScript.
