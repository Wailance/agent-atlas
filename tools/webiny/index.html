О проекте

webiny — aPI-first CMS, контентные модели и headless-управление сайтом. Ключевые темы: Mongodb, Php, Platform, React, Webiny и Webpack. Проект: есть бесплатный tier и платные возможности и основной стек — JavaScript.

webiny: [NOT MAINTAINED] Main Webiny app responsible for parsing requests and bootstrapping other apps. Includes all the core ReactJs components. In this catalog (Business), listed as “Headless CMS”: API-first CMS, content models and headless content management. Key topics: Mongodb, Php, Platform, React, Webiny, and Webpack. Project details: freemium model and built mainly in JavaScript.
