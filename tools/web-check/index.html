О проекте

Web Check — собирает данные с сайтов, PDF и документов — вручную или с помощью ИИ. Можно развернуть на своём сервере. Ключевые темы: Osint, Безопасность, Свой сервер, Приватность, Security Tools и Sysadmin. Проект: бесплатный open-source проект, на GitHub 34k+ звёзд и основной стек — TypeScript. Поддерживает ИИ-агентов. Открытый исходный код.

Web Check: 🕵️‍♂️ All-in-one OSINT tool for analysing any website. In this catalog (AI & Agents), listed as “Web Scraping”: Web data collection, crawlers, AI scraping, document parsing. Key topics: Osint, Security, на своём сервере, Privacy, Security Tools, and Sysadmin. Project details: free с открытым кодом, 34k+ GitHub stars, and built mainly in TypeScript.

В каталоге отнесён к разделам: «Парсинг и сбор данных» — собирает данные с сайтов, PDF и документов — вручную или с помощью ИИ; «Аналитика и отчёты» — графики, дашборды и отчёты по данным бизнеса и сайта.

На GitHub у проекта более 34k+ звёзд — это один из заметных репозиториев в своей нише.
