О проекте

MetaRadar — другие полезные open-source проекты. Открытый исходный код.

MetaRadar is a tool for BLE environment monitoring. Find and track Bluetooth devices around, and get notified when the target device is detected. In this catalog (Infrastructure), listed as “General OSS”: General с открытым кодом software. Key topics: Android, Android Ble, Ble, Bluetooth, Radar, and Scanner. Project details: free с открытым кодом and built mainly in Kotlin.
