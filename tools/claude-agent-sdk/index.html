О проекте

Claude Agent SDK — помогает писать, исправлять и объяснять код в редакторе или терминале. Работает с ИИ-агентами. Ключевые темы: Anthropic, MCP и Hooks. Проект: бесплатный с открытым кодом проект, на GitHub 8k+ звёзд и основной стек — Python. Работает с документами и файлами.

Claude Agent SDK: sDK for building agents on Claude Code architecture: file/shell access, MCP, lifecycle hooks. In this catalog (AI & Agents), listed as “Coding Agents”: Agents for writing and editing code in IDE and terminal. Key topics: Anthropic, MCP, and Hooks. Project details: free с открытым кодом, 8k+ GitHub stars, and built mainly in Python.

В каталоге отнесён к разделам: «ИИ для кода» — помогает писать, исправлять и объяснять код в редакторе или терминале; «Оркестрация агентов» — запускает несколько ИИ-агентов и связывает их в цепочки задач.

На GitHub у проекта более 8k+ звёзд — это один из заметных репозиториев в своей нише.
