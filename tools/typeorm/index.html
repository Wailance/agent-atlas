О проекте

typeorm — клиенты API, шлюзы, webhooks и обмен данными между системами. Связывается с другими сервисами. Открытый исходный код.

typeorm: typeScript & JavaScript ORM for Node.js — supports PostgreSQL, MySQL, MariaDB, SQLite, SQL Server, Oracle, and more. In this catalog (Infrastructure), listed as “API & интеграции”: API clients, gateways, webhooks, ETL, data pipelines. Key topics: Active Record, Cockroachdb, Data Mapper, Database, Mariadb, and Mysql. Project details: free с открытым кодом, 37k+ GitHub stars, and built mainly in TypeScript.

В каталоге отнесён к разделам: «API и интеграции» — клиенты API, шлюзы, webhooks и обмен данными между системами; «Базы данных» — хранение данных, SQL-клиенты и админки баз.

На GitHub у проекта более 37k+ звёзд — это один из заметных репозиториев в своей нише.
