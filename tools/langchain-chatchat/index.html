О проекте

Langchain-Chatchat — подключает ваши файлы и базы к чату с ИИ, чтобы ответы были точнее. Работает с ИИ-агентами, обрабатывает документы и контент для ИИ. Ключевые темы: Чат-бот, Chatchat, Chatglm, Chatgpt, Embedding и Faiss. Подходит, если нужно: работает с ИИ-агентами, обрабатывает документы и контент для ИИ и можно запускать локально. Проект: бесплатный open-source проект, на GitHub 38k+ звёзд и основной стек — Python. Открытый исходный код.

Langchain-Chatchat: langchain-Chatchat（原Langchain-ChatGLM）基于 Langchain 与 ChatGLM, Qwen 与 Llama 等语言模型的 поиск по документам 与 Agent 应用 | Langchain-Chatchat (formerly langchain-ChatGLM), local knowledge based языковые модели (like ChatGLM, Qwen and Llama) поиск по документам and Agent app with langchain. In this catalog (AI & Agents), listed as “поиск по документам / Context”: поиск по документам, memory and context for языковые модели. Key topics: Chatbot, Chatchat, Chatglm, Chatgpt, Embedding, and Faiss. Useful when you need to build AI agents and work with documents and поиск по документам. Project details: free с открытым кодом, 38k+ GitHub stars, and built mainly in Python.
