О проекте

bonita-engine — bPM, durable execution и движки для бизнес- и системных процессов

bonita-engine: deploy, execute, manage process-based applications made with Bonita studio or through Engine APIs. In this catalog (Infrastructure), listed as “рабочие процессы Engines”: BPM, durable execution and engines for business and system рабочие процессыs. Key topics: Bpm, Bpm Engine, Bpmn, Delivery Central, Engine, and Lifecycle Platform. Project details: freemium model and built mainly in Java.

В каталоге отнесён к разделам: «Процессные движки» — bPM, durable execution и движки для бизнес- и системных процессов; «Внутренние сервисы» — админки, панели и внутренние приложения для команды без долгой разработки.
