О проекте

rundeck — связывает сервисы и автоматизирует рутину — как Zapier, но у себя. Автоматизирует процессы и связывает сервисы. Ключевые темы: Ansible, Audit, Автоматизация, Category Distributed, Deployment и DevOps. Проект: бесплатный с открытым кодом проект, на GitHub 6k+ звёзд и основной стек — Groovy.

rundeck: enable Self-Service Operations: Give specific users access to your existing tools, services, and scripts. In this catalog (Business), listed as “автоматизация & рабочие процессыs”: рабочие процессы автоматизация, интеграции, RPA, process orchestration. Key topics: Ansible, Audit, автоматизация, Category Distributed, Deployment, and DevOps. Project details: free с открытым кодом, 6k+ GitHub stars, and built mainly in Groovy.
