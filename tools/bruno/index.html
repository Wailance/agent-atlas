О проекте

bruno — клиенты API, шлюзы, webhooks и обмен данными между системами. Автоматизирует процессы и связывает сервисы. Ключевые темы: Api Client, Api Testing, Автоматизация, Для разработчиков, Git и Graphql Client. Проект: бесплатный с открытым кодом проект, на GitHub 46k+ звёзд и основной стек — JavaScript.

bruno: opensource IDE For Exploring and Testing API's (lightweight alternative to Postman/Insomnia). In this catalog (Infrastructure), listed as “API & интеграции”: API clients, gateways, webhooks, ETL, data pipelines. Key topics: Api Client, Api Testing, автоматизация, Developer tools, Git, and Graphql Client. Project details: free с открытым кодом, 46k+ GitHub stars, and built mainly in JavaScript.
