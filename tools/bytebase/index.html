О проекте

bytebase — хранение данных, SQL-клиенты и админки баз. Ключевые темы: Cicd, Data Masking, Data Security, Database Access, Dbeaver и Devsecops. Проект: есть бесплатный tier и платные возможности, на GitHub 14k+ звёзд и основной стек — Go.

bytebase: world's most advanced database DevSecOps solution for Developer, Security, DBA and Platform Engineering teams. The GitHub/GitLab for database DevSecOps. In this catalog (Infrastructure), listed as “Databases & Admin”: SQL clients, database admin, vector DB, time-series. Key topics: Cicd, Data Masking, Data Security, Database Access, Dbeaver, and Devsecops. Project details: freemium model, 14k+ GitHub stars, and built mainly in Go.
