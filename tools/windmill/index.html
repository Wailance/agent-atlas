О проекте

windmill — связывает сервисы и автоматизирует рутину — как Zapier, но у себя. Автоматизирует процессы и связывает сервисы. Ключевые темы: Мало кода, Platform, PostgreSQL и Self Hostable. Проект: есть бесплатный tier и платные возможности, на GitHub 17k+ звёзд и основной стек — Rust. Открытый исходный код.

windmill: с открытым кодом developer platform to power your entire infra and turn scripts into webhooks, рабочие процессыs and UIs. Fastest рабочие процессы engine (13x vs Airflow). с открытым кодом alternative to Retool and Temporal. In this catalog (Business), listed as “автоматизация & рабочие процессыs”: рабочие процессы автоматизация, интеграции, RPA, process orchestration. Key topics: Low-code, Platform, PostgreSQL, and Self Hostable. Project details: freemium model, 17k+ GitHub stars, and built mainly in Rust.

В каталоге отнесён к разделам: «Автоматизация» — связывает сервисы и автоматизирует рутину — как Zapier, но у себя; «Внутренние сервисы» — админки, панели и внутренние приложения для команды без долгой разработки.

На GitHub у проекта более 17k+ звёзд — это один из заметных репозиториев в своей нише.
