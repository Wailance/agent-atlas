О проекте

decap-cms — блоги, лендинги и управление контентом сайта. Открытый исходный код.

decap-cms is a Git-based CMS for Static Site Generators. In this catalog (Business), listed as “Websites & CMS”: CMS, headless CMS, blogs, site builders, landing pages. Key topics: CMS, Content Management System, Decap Cms, Git Based Cms, Headless CMS, and Jamstack. Project details: free с открытым кодом, 19k+ GitHub stars, and built mainly in JavaScript.
