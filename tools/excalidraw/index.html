О проекте

excalidraw — база знаний, заметки и совместная работа с документами. Ключевые темы: Canvas, Совместная работа, Diagrams, Drawing, Продуктивность и Whiteboard. Проект: бесплатный с открытым кодом проект, на GitHub 127k+ звёзд и основной стек — TypeScript.

excalidraw: virtual whiteboard for sketching hand-drawn like diagrams. In this catalog (Business), listed as “Documents & Wiki”: Wiki, knowledge base, document management, Notion alternatives. Key topics: Canvas, Collaboration, Diagrams, Drawing, Productivity, and Whiteboard. Project details: free с открытым кодом, 127k+ GitHub stars, and built mainly in TypeScript.
