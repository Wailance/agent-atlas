О проекте

DocumentServer — таблицы и простые базы данных для команды — аналог Airtable. Обрабатывает документы и контент для ИИ. Ключевые темы: Совместная работа, Collaborative, Doc, Docx, Excel и Node. Проект: бесплатный с открытым кодом проект, на GitHub 7k+ звёзд и основной стек — Shell.

DocumentServer: oNLYOFFICE Docs is a free collaborative online office suite comprising viewers and editors for texts, spreadsheets and presentations, forms and PDF, fully compatible with Office Open XML formats: .docx, .xlsx, .pptx and enabling collaborative editing in real time. In this catalog (Business), listed as “Spreadsheets & Data”: Airtable alternatives, spreadsheets, data grids, no-code databases. Key topics: Collaboration, Collaborative, Doc, Docx, Excel, and Node. Project details: free с открытым кодом, 7k+ GitHub stars, and built mainly in Shell.
