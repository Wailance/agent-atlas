О проекте

timescaledb — хранение данных, SQL-клиенты и админки баз. Подходит для голосовых сценариев. Ключевые темы: Аналитика, База данных, Financial Analysis, Iot, Postgres и PostgreSQL. Проект: есть бесплатный tier и платные возможности, на GitHub 23k+ звёзд и основной стек — C.

timescaledb is a time-series database for high-performance real-time analytics packaged as a Postgres extension. In this catalog (Infrastructure), listed as “Databases & Admin”: SQL clients, database admin, vector DB, time-series. Key topics: Analytics, Database, Financial Analysis, Iot, Postgres, and PostgreSQL. Project details: freemium model, 23k+ GitHub stars, and built mainly in C.

В каталоге отнесён к разделам: «Базы данных» — хранение данных, SQL-клиенты и админки баз; «Аналитика и отчёты» — графики, дашборды и отчёты по данным бизнеса и сайта.

На GitHub у проекта более 23k+ звёзд — это один из заметных репозиториев в своей нише.
