О проекте

airbyte — клиенты API, шлюзы, webhooks и обмен данными между системами. Можно развернуть на своём сервере, работает с ИИ-агентами. Ключевые темы: Bigquery, Change Data Capture, Data, Data Analysis, Data Collection и Data Engineering. Подходит, если нужно: можно развернуть на своём сервере и работает с ИИ-агентами. Проект: есть бесплатный tier и платные возможности, на GitHub 22k+ звёзд и основной стек — Python. Работает с документами и файлами. Связывается с другими сервисами. Открытый исходный код.

airbyte: с открытым кодом data movement for ELT pipelines and AI agents — from APIs, databases & files to warehouses, lakes, and AI applications. Both на своём сервере and Cloud. In this catalog (Infrastructure), listed as “API & интеграции”: API clients, gateways, webhooks, ETL, data pipelines. Key topics: Bigquery, Change Data Capture, Data, Data Analysis, Data Collection, and Data Engineering. Useful when you need to self-host on your infrastructure and build AI agents. Project details: freemium model, 22k+ GitHub stars, and built mainly in Python.

В каталоге отнесён к разделам: «API и интеграции» — клиенты API, шлюзы, webhooks и обмен данными между системами; «Автоматизация» — связывает сервисы и автоматизирует рутину — как Zapier, но у себя.

На GitHub у проекта более 22k+ звёзд — это один из заметных репозиториев в своей нише.
