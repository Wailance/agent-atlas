О проекте

Mayan-EDMS — база знаний, заметки и совместная работа с документами. Обрабатывает документы и контент для ИИ. Ключевые темы: Django, Dms, Документы, Document Management System, Document Manager и Enterprise. Проект: бесплатный с открытым кодом проект и основной стек — Python.

Mayan-EDMS: free Open Source Document Management System (mirror, no pull request or issues). In this catalog (Business), listed as “Documents & Wiki”: Wiki, knowledge base, document management, Notion alternatives. Key topics: Django, Dms, Documents, Document Management System, Document Manager, and Enterprise. Project details: free с открытым кодом and built mainly in Python.
