О проекте

Claudegram — управление задачами и ИИ прямо из Telegram. Поддерживает ИИ-агентов. Открытый исходный код.

Claudegram: telegram bot that bridges messages to Claude Code. In this catalog (AI & Agents), listed as “Telegram Bridges”: Control coding agents and AI via Telegram. Key topics: Telegram, Claude Code, and Streaming. Project details: free с открытым кодом and built mainly in TypeScript.

В каталоге отнесён к разделам: «Telegram и боты» — управление задачами и ИИ прямо из Telegram; «ИИ для кода» — помогает писать, исправлять и объяснять код в редакторе или терминале.
