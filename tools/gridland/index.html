О проекте

Gridland — помогает писать, исправлять и объяснять код в редакторе или терминале. Ключевые темы: React, Terminal и Tui. Проект: бесплатный с открытым кодом проект и основной стек — TypeScript. Поддерживает ИИ-агентов.

Gridland: terminal apps that run anywhere - even the browser! In this catalog (AI & Agents), listed as “Coding Agents”: Agents for writing and editing code in IDE and terminal. Key topics: React, Terminal, and Tui. Project details: free с открытым кодом and built mainly in TypeScript.
