О проекте

marker — собирает данные с сайтов, PDF и документов — вручную или с помощью ИИ. Обрабатывает документы и контент для ИИ. Проект: бесплатный open-source проект, на GitHub 37k+ звёзд и основной стек — Python. Поддерживает ИИ-агентов. Открытый исходный код.

marker: convert PDF to markdown + JSON quickly with high accuracy. In this catalog (AI & Agents), listed as “Web Scraping”: Web data collection, crawlers, AI scraping, document parsing. Project details: free с открытым кодом, 37k+ GitHub stars, and built mainly in Python.

В каталоге отнесён к разделам: «Парсинг и сбор данных» — собирает данные с сайтов, PDF и документов — вручную или с помощью ИИ; «Поиск по документам» — подключает ваши файлы и базы к чату с ИИ, чтобы ответы были точнее.

На GitHub у проекта более 37k+ звёзд — это один из заметных репозиториев в своей нише.
