О проекте

gateway — один вход для разных ИИ-моделей: маршрутизация, лимиты, ключи. Автоматизирует процессы и связывает сервисы. Ключевые темы: Ai Gateway, Gateway, Generative Ai, Langchain, Языковые модели и Llm Gateway. Проект: есть бесплатный tier и платные возможности, на GitHub 12k+ звёзд и основной стек — TypeScript. Поддерживает ИИ-агентов.

gateway is a blazing fast AI Gateway with integrated guardrails. Route to 1,600+ языковые модели, 50+ AI Guardrails with 1 fast & friendly API. In this catalog (AI & Agents), listed as “языковые модели Gateways”: Routing, proxies and unified API for языковые модели providers. Key topics: Ai Gateway, Gateway, Generative Ai, Langchain, языковые модели, and Llm Gateway. Project details: freemium model, 12k+ GitHub stars, and built mainly in TypeScript.
