О проекте

nanobrowser — связывает сервисы и автоматизирует рутину — как Zapier, но у себя. Автоматизирует процессы и связывает сервисы, работает с ИИ-агентами. Ключевые темы: Агент, ИИ, ИИ-агенты, Ai Tools, Автоматизация и Browser. Подходит, если нужно: автоматизирует процессы и связывает сервисы и работает с ИИ-агентами. Проект: бесплатный open-source проект, на GitHub 13k+ звёзд и основной стек — TypeScript. Открытый исходный код.

nanobrowser: с открытым кодом Chrome extension for AI-powered web автоматизация. Run несколько ИИ-агентов рабочие процессыs using your own языковые модели API key. Alternative to OpenAI Operator. In this catalog (Business), listed as “автоматизация & рабочие процессыs”: рабочие процессы автоматизация, интеграции, RPA, process orchestration. Key topics: Agent, AI, AI agents, Ai Tools, автоматизация, and Browser. Useful when you need to automate рабочие процессыs and build AI agents. Project details: free с открытым кодом, 13k+ GitHub stars, and built mainly in TypeScript.

В каталоге отнесён к разделам: «Автоматизация» — связывает сервисы и автоматизирует рутину — как Zapier, но у себя; «Парсинг и сбор данных» — собирает данные с сайтов, PDF и документов — вручную или с помощью ИИ.

На GitHub у проекта более 13k+ звёзд — это один из заметных репозиториев в своей нише.
