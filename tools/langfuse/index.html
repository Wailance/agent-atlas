О проекте

Langfuse — следит за качеством ответов, ошибками и расходами ИИ-приложений. Автоматизирует процессы и связывает сервисы. Ключевые темы: Tracing, Eval, Наблюдаемость, Аналитика, Autogen и Evaluation. Проект: есть бесплатный tier и платные возможности, на GitHub 31k+ звёзд и основной стек — TypeScript. Поддерживает ИИ-агентов. Открытый исходный код.

Langfuse: 🪢 Open source AI engineering platform: языковые модели evals, observability, metrics, prompt management, playground, datasets. Integrates with OpenTelemetry, LangChain, OpenAI SDK, LiteLLM, and more. 🍊YC W23. In this catalog (AI & Agents), listed as “Observability / Eval”: Tracing, evaluation, monitoring of AI applications. Key topics: Tracing, Eval, Observability, Analytics, Autogen, and Evaluation. Project details: freemium model, 31k+ GitHub stars, and built mainly in TypeScript.

В каталоге отнесён к разделу: «Мониторинг ИИ» — следит за качеством ответов, ошибками и расходами ИИ-приложений.
