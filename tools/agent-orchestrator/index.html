О проекте

Agent Orchestrator — запускает несколько ИИ-агентов и связывает их в цепочки задач. Работает с ИИ-агентами. Ключевые темы: Orchestration, Ci, Multi Agent, Agent Fleet, Agent Swarm и Claude Code. Проект: бесплатный с открытым кодом проект, на GitHub 8k+ звёзд и основной стек — Go. Связывается с другими сервисами.

Agent Orchestrator: agentic orchestrator for parallel coding agents — plans tasks, spawns agents, and autonomously handles CI fixes, merge conflicts, and code reviews. In this catalog (AI & Agents), listed as “Agent Harness”: Orchestration, несколько ИИ-агентов рабочие процессыs, long-horizon tasks. Key topics: Orchestration, Ci, Multi Agent, Agent Fleet, Agent Swarm, and Claude Code. Project details: free с открытым кодом, 8k+ GitHub stars, and built mainly in Go.

В каталоге отнесён к разделам: «Оркестрация агентов» — запускает несколько ИИ-агентов и связывает их в цепочки задач; «ИИ для кода» — помогает писать, исправлять и объяснять код в редакторе или терминале.

На GitHub у проекта более 8k+ звёзд — это один из заметных репозиториев в своей нише.
