О проекте

bigbluebutton — чат, видеозвонки и форумы для команды — альтернатива Slack. Открытый исходный код.

bigbluebutton is a complete web conferencing system for virtual classes and more! In this catalog (Infrastructure), listed as “Team Communication”: Team chat, video, forums — Slack/Discord alternatives. Key topics: Bigbluebutton, Collaboration, Conferencing, Video, Video Conferencing, and Webrtc. Project details: free с открытым кодом, 9k+ GitHub stars, and built mainly in JavaScript.
