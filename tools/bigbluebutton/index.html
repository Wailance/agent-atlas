О проекте

bigbluebutton — видеозвонки, веб-конференции, WebRTC и realtime-коммуникации. Открытый исходный код.

bigbluebutton is a complete web conferencing system for virtual classes and more! In this catalog (Infrastructure), listed as “Video Conferencing”: Video calls, web conferencing, WebRTC and realtime communication stacks. Key topics: Bigbluebutton, Collaboration, Conferencing, Video, Video Conferencing, and Webrtc. Project details: free с открытым кодом, 9k+ GitHub stars, and built mainly in JavaScript.
